Claude Code
+ qmd in 2 minutes.

One command to set up Claude Code with Tobi Lütke's on-device hybrid search over a local markdown workspace. Zero config, no API keys, no MCP wiring. Built for operators who want to skip the basics.

curl -fsSL https://raw.githubusercontent.com/demi-hl/create-mcp-workspace/main/install.sh | bash

Paste in your terminal. macOS, Linux, or WSL.

What gets installed

Claude Code

Anthropic's terminal-native AI coding agent.

qmd

On-device BM25 + vector + LLM-rerank search over markdown. Auto-wires its own MCP.

~/claude-workspace

Folder seeded with CLAUDE.md, MEMORY.md, and a first-session interview prompt.

Before you run

  1. Node 18+ — nodejs.org, or brew install node
  2. Git — preinstalled on most systems
  3. An Anthropic account — Claude Code prompts for login on first run

After it finishes

cd ~/claude-workspace
claude

Paste the First Session Prompt from your new CLAUDE.md. Claude interviews you for your role, projects, and preferences, and seeds memory so future sessions know you.
